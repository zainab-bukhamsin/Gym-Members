# Gym Members Exercise Dataset - Data Preprocessing 

## Dataset Source
This dataset was obtained from Kaggle: 
https://www.kaggle.com/datasets/valakhorasani/gym-members-exercise-dataset

## Dataset Overview
- Number of rows: 973
- Number of columns: 15

## Features Description

- Age: Age of the gym member in years
- Gender: Gender of the member (Male / Female)
- Weight (kg): Body weight in kilograms
- Height (m): Height in meters
- BMI: Body Mass Index calculated from weight and height
- Fat_Percentage: Body fat percentage
- Max_BPM: Maximum heart rate during workout (beats per minute)
- Avg_BPM: Average heart rate during workout (beats per minute)
- Resting_BPM: Resting heart rate (beats per minute)
- Session_Duration (hours): Duration of each workout session in hours
- Calories_Burned: Total calories burned per session
- Water_Intake (liters): Daily water intake in liters
- Workout_Frequency (days/week): Number of workout days per week
- Workout_Type: Type of workout performed (e.g., Cardio, Strength, Yoga, HIIT)
- Experience_Level: Fitness experience level of the member (1 = Beginner, 3 = Expert)

## Purpose of the Assignment
The purpose of this dataset is to perform Data Quality Assessment & Preprocessing to understand:
- How to detect and fix data type inconsistencies in real-world fitness data
- How to handle missing values using appropriate imputation strategies
- How to detect and remove outliers using the IQR method
- How to normalize and standardize numerical features for machine learning
- How to apply PCA for dimensionality reduction and feature analysis
